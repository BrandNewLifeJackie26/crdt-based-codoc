# crdt-based-codoc

to run this web-assembly:

### under ./wasm_client
1. run ```wasm-pack build```. Only need to run this if you modify the rust source code
2. if ./wasm_client/www not exist, run  ```npm init wasm-app www``` 

### under ./wasm_client/www
1. run ```npm install```
2. run ```npm run start```
